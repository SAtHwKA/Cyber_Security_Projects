# 🔐 Cybersecurity Projects

This repository contains 3 beginner-level cybersecurity projects implemented using Python. These projects demonstrate core security concepts such as password strength analysis, encryption, and phishing detection.

---

## 📌 Project 1: Password Strength Checker
This program evaluates the strength of a password based on:
- Length
- Uppercase and lowercase letters
- Numbers
- Special characters

🔍 Output:
- Weak ❌
- Moderate ⚠
- Strong ✅

---

## 📌 Project 2: Caesar Cipher (Encryption & Decryption)
This tool allows users to:
- Encrypt messages using a shift value
- Decrypt messages using the same shift

🔐 Concept:
- Each letter is shifted forward/backward in the alphabet.

---

## 📌 Project 3: Phishing Awareness Analyzer
This program detects potential phishing messages using:

### 🚨 Detection Techniques:
- Urgency words (e.g., "urgent", "act now")
- Fake domains (e.g., gooogle.com)
- Social engineering (e.g., "send me OTP")
- Suspicious links
- Context-based OTP detection

🧠 It intelligently distinguishes between:
- Legitimate OTP messages ✅  
- Phishing attempts ❌
